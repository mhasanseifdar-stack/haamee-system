# 🚀 سامانه Haamee - آماده تحویل

## ⚡ شروع سریع (5 دقیقه)

### 1️⃣ نصب Backend

```bash
cd haamee-backend
npm install
npm start
```

### 2️⃣ نصب Frontend (ترمینال جدید)

```bash
cd haamee-frontend
npm install
npm start
```

### 3️⃣ ورود

مرورگر خودکار باز می‌شود: http://localhost:3000

- **نام کاربری:** admin
- **رمز عبور:** 123456
